# Get Opened Note

This skill returns the meeting note that is currently open in the Granola Notes UI, including the title, formatted note content, and transcript (if available).

## Usage

- No arguments required.
- The skill reads the currently viewed meeting from the UI. If no meeting is open, all fields will be empty strings.
